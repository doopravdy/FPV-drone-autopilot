# FPV-drone-autopilot
穿越机纯视觉端到端自动驾驶开源项目（开发中）
## 效果体验/使用方法：
- 下载 ViGEm Bus Driver，并虚拟一个python环境（只有一个代码，可自行下载缺失的第三方库。不建议使用电脑纯python环境）
- 设置config.json,使用.pth文件，掩膜maks.png（可选）
- 运行autopilot.py，即查看效果
- (代码有点粗糙，仅验证项目能跑，但效果不好。原因有数据集，（路线:CNN + LSTM），代码本身bug）

### 选择模式：
- 手动模式 /自动模式
- 穿越机erls2.4g版本遥控器CH 7 ：00 00（是自动模式）FF 03(是手动模式）（已知适配遥控器：小白控3代，右手拨片）
#### 数据收集工具（代码仅供参考）：
- 这个文件是数据收集的一个思路，期望更多人参与本项目（穿越机自动驾驶仪-只靠图传画面）。
- 穿越机遥控器（小白控，右手边按钮）CH 8 ：00 00 (录制）/ FF 03(结束）
